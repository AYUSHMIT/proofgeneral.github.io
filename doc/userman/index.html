Proof General

This file documents version 4.4.1~pre of Proof General, a generic Emacs interface for proof assistants.

Proof General 4.4.1~pre has been tested with GNU Emacs 24.4 on Linux. It is supplied ready to use for the proof assistants LEGO, Coq, Isabelle, and PhoX. Experimental support is provided for several other provers.
