Proof General

This file documents version 4.5-git of Proof General, a generic Emacs interface for proof assistants.

Proof General 4.5-git has been tested with GNU Emacs 24.3 on Linux. It is supplied ready to use for the proof assistants Coq, EasyCrypt, and PhoX.
